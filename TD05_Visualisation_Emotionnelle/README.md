# TD05 : Visualisation des données - Obtenir une réponse émotionnelle

## Objectifs du TD
- Utiliser la couleur et le design pour créer de l'impact
- Réduire la charge cognitive
- Appliquer le principe "Moins c'est plus"
- Créer des dashboards engageants et efficaces

## Contenu
- Théorie des couleurs et psychologie visuelle
- Design minimaliste et clarté
- Éliminer le bruit visuel
- Créer de l'engagement avec les données
- Storytelling avec les données

## Fichiers
- `notebook.ipynb` : Notebook interactif Google Colab
- `presentations/` : Slides générées depuis NotebookLM
- `images/` : Infographies générées depuis NotebookLM
- `resources/` : Fichiers de données et ressources du TD

## Comment utiliser ce TD
1. Ouvrir le notebook dans Google Colab : [Lien Colab](https://colab.research.google.com/github/Maxencegu/power_bi_course_upjv/blob/main/TD05_Visualisation_Emotionnelle/notebook.ipynb)
2. Consulter les slides et infographies pour la théorie
3. Suivre les exercices du notebook et les cas d'étude
4. Compléter le questionnaire disponible dans Google Classroom

## Ressources
- Les fichiers de données et ressources supplémentaires sont dans le dossier `resources/`
