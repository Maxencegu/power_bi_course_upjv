# Power BI Course - UPJV

Matériaux de cours Power BI pour l'Université Picardie Jules Verne (UPJV).

## 📚 Structure du cours

Le cours est organisé en **5 travaux dirigés (TDs)** de 2 heures chacun, plus **1 travail à la maison** en binôme.

### Travaux Dirigés (en classe)

| TD | Thème | Contenu |
|----|-------|---------|
| **TD01** | Premiers pas + Transformer les données | Interface, chargement, Éditeur de requêtes, nettoyage |
| **TD02** | Visualiser et Filtrer | Visualisations, filtres, slicers, interactions |
| **TD03** | DAX et Contexte | Fonctions DAX, contexte, mesures, colonnes calculées |
| **TD04** | DAX avec les dates + Visualisation | Analyses temporelles, principes de design |
| **TD05** | Visualisation émotionnelle | Couleurs, design minimaliste, storytelling |

### Travaux à la maison (en binôme)

- **Étude de cas : Analyse du taux de désabonnement** : Analysez les patterns de churn clients et créez un rapport Power BI avec recommandations

## 🚀 Comment utiliser ce cours

### Pour les étudiants

1. **Accéder aux notebooks** :
   - Chaque TD contient un lien direct pour ouvrir le notebook dans Google Colab
   - Voir le README.md de chaque TD pour le lien

2. **Ressources** :
   - Les slides (PDF) et infographies (PNG) générées depuis NotebookLM se trouvent dans les dossiers `presentations/` et `images/`
   - Chaque TD inclut un README.md avec ses objectifs et contenus

3. **Présence et Questionnaire** :
   - Les TDs se font **en présentiel**
   - Feuille de présence Google Forms à remplir en classe
   - Questionnaire disponible dans Google Classroom

4. **Travail à la maison** :
   - À faire **en binôme**
   - À rendre **avant la date limite** communiquée
   - Voir les détails dans le README.md du travail
   - Format : Fichier Power BI (.pbix) + documentation

### Pour le professeur

- Ajouter les notebooks `.ipynb` dans chaque dossier TD
- Mettre à jour les dossiers `resources/` avec les fichiers de données Power BI
- Configurer Google Classroom avec les liens et questionnaires
- Gérer les Google Forms pour la présence et les questionnaires

## 📊 Plateforme d'apprentissage

- **GitHub** : Stockage du contenu
- **Google Colab** : Démonstrations et explications dans les notebooks
- **Power BI Desktop** : Environnement principal pour les pratiques
- **Google Classroom** : Interface principale pour les étudiants
- **Google Forms** : Présence et questionnaires
- **Google Sheets** : Suivi des résultats
