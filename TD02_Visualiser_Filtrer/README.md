# TD02 : Visualiser des données + Filtrer

## Objectifs du TD
- Créer des visualisations efficaces dans Power BI
- Ajouter des filtres et des slicers
- Comprendre le modèle de données
- Utiliser les relations entre tables

## Contenu
- Types de visualisations dans Power BI
- Créer et configurer des graphiques
- Ajouter des filtres et des slicers
- Filtres au niveau visuel, page et rapport
- Interactions entre visualisations

## Fichiers
- `notebook.ipynb` : Notebook interactif Google Colab
- `presentations/` : Slides générées depuis NotebookLM
- `images/` : Infographies générées depuis NotebookLM
