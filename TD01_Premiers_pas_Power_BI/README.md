# TD01 : Premiers pas avec Power BI + Transformer les données

## Objectifs du TD
- Découvrir l'interface de Power BI
- Importer et charger des données
- Comprendre et utiliser l'Éditeur de requêtes
- Transformer et nettoyer les données

## Contenu
- Introduction à Power BI Desktop
- Connexion et chargement de données
- Vue d'ensemble de l'interface
- Transformation de données dans l'Éditeur de requêtes
- Nettoyage et préparation des données

## Fichiers
- `notebook.ipynb` : Notebook interactif Google Colab (démonstrations et exercices)
- `presentations/` : Slides générées depuis NotebookLM
- `images/` : Infographies générées depuis NotebookLM
- `resources/` : Fichiers de données et ressources du TD

## Comment utiliser ce TD
1. Ouvrir le notebook dans Google Colab : [Lien Colab](https://colab.research.google.com/github/Maxencegu/power_bi_course_upjv/blob/main/TD01_Premiers_pas_Power_BI/notebook.ipynb)
2. Consulter les slides et infographies pour la théorie
3. Suivre les exercices du notebook
4. Compléter le questionnaire disponible dans Google Classroom

## Ressources
- Les fichiers de données et ressources supplémentaires sont dans le dossier `resources/`
- Données d'exemple pour pratiquer les transformations
